# vue-cli_project

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Router
```
npm install vue-router --save
```

### VUEX
```
npm install vuex --save
```


### VUE-carousel
```
npm install -S vue-carousel
```
